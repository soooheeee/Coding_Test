### 문제 설명

<p>알파벳 소문자로만 이루어진 단어가 주어진다. 이때, 이 단어가 팰린드롬인지 아닌지 확인하는 프로그램을 작성하시오.</p>

<p>팰린드롬이란 앞으로 읽을 때와 거꾸로 읽을 때 똑같은 단어를 말한다. </p>

<p>level, noon은 팰린드롬이고, baekjoon, online, judge는 팰린드롬이 아니다.</p>

### 입력 

 <p>첫째 줄에 단어가 주어진다. 단어의 길이는 1보다 크거나 같고, 100보다 작거나 같으며, 알파벳 소문자로만 이루어져 있다.</p>

### 출력 

 <p>첫째 줄에 팰린드롬이면 1, 아니면 0을 출력한다.</p>


# 아이디어 및 기능
입력된 문자열을 역순으로 봤을때, 원래 입력값이랑 같은지 비교하기 맞다면 1
아니라면 0으로 출력

역순 확인 방법
1. [::-1] 리스트에서 슬라이싱 기능을 이용하면 역순으로 값을 비교할 수 있다 내가 푼 방법도 슬라이싱 기능을 이용해서 푼 것이다.
2. reverse()이용하기
이번에 문제를 풀면서 reverse랑 reversed에 대한 차이점이 궁금해졌다
그래서 찾아본 결과
둘 다 파이썬에서 리스트의 요소를 뒤집을 때 사용하는 기능이다
차이점은
# reverse
reverse는 list타입에서 제공
# reversed
reversed는 내장함수로, list에서 제공하는 함수가 아니다.
만약 역순으로 사용하고 싶다면 
l = ['a', 'b', 'c']
t = ('a', 'b', 'c')

list(reversed(l))  # ['c', 'b', 'a']
tuple(reversed(t))  # ('c', 'b', 'a')
이런식으로 사용해야한다.
